Тест переноса слов

Тест с длинным текстом в узком контейнере:

ШиноСервис Экспресс на Оболоні очень длинное название сервисной точки
Коваленко Олег Васильевич длинное имя клиента
+38 (661) 234-56-78 телефон

Проверка ключей переводов

Ключи, которые должны переводиться:

admin:orders.title → "Заказы интернет-магазинов"
admin:orders.searchPlaceholder → "Поиск по ТТН, имени клиента или телефону..."
admin:orders.columns.status → "Статус"
admin:orders.columns.customer → "Клиент"
admin:orders.thirdParty → "Сторонняя"
common:total → "Всего"
common:notSpecified → "Не указано"
common:units.pieces → "шт."

Инструкции для проверки

  1. Открыть страницу /admin/store-orders в браузере
  2. Открыть консоль разработчика (F12)
  3. Проверить наличие ошибок в консоли
  4. В консоли выполнить: localStorage.getItem('i18nextLng') для проверки языка
  5. Проверить загрузку файлов переводов в Network tab
  6. Переключить язык и проверить изменения

Возможные причины проблем

  • Файлы переводов не перезагрузились в браузере (нужна очистка кэша)
  • Сервер разработки не перезапущен после изменения файлов переводов
  • Ошибка в структуре JSON файлов переводов
  • Неправильная конфигурация i18next
  • CSS стили не применяются из-за специфичности селекторов

🔧 Настройка Google Cloud Console

Статус: Требуется настройка

Шаги настройки:

  1. Перейти в Google Cloud Console
  2. Создать новый проект или выбрать существующий
  3. Включить Google+ API
  4. Создать OAuth 2.0 Client ID
  5. Настроить Authorized redirect URIs
Authorized redirect URIs: http://localhost:8000/auth/google/callback http://localhost:8000/auth/google/failure https://yourdomain.com/auth/google/callback (для продакшена)
🔧 Открыть Google Cloud Console

🌐 Переменные окружения

Добавьте следующие переменные в .env файл:

# Google OAuth настройки GOOGLE_CLIENT_ID=your_google_client_id_here GOOGLE_CLIENT_SECRET=your_google_client_secret_here FRONTEND_URL=http://localhost:3008

🐛 Возможные проблемы и решения

❌ Ошибка: "OAuth данные не получены"

  • Проверьте настройки Google Cloud Console
  • Убедитесь, что redirect URI правильный
  • Проверьте GOOGLE_CLIENT_ID и GOOGLE_CLIENT_SECRET

❌ Ошибка: "Роль client не найдена"

  • Выполните seeds: rails db:seed
  • Проверьте наличие роли в таблице user_roles

❌ Ошибка CORS

  • Проверьте настройки CORS в config/initializers/cors.rb
  • Убедитесь, что frontend URL добавлен в origins

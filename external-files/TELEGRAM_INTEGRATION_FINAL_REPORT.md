### База данных

**Таблицы:**
- `telegram_subscriptions` - Подписки пользователей
- `telegram_notifications` - История уведомлений

**Связи:**
- User → TelegramSubscription (1:many)
- TelegramSubscription → TelegramNotification (1:many)
- Booking → TelegramNotification (1:many)

### API эндпоинты

**Подписки:**
- `GET /api/v1/telegram_subscriptions` - Список подписок
- `POST /api/v1/telegram_subscriptions` - Создание подписки
- `PATCH /api/v1/telegram_subscriptions/:id` - Обновление настроек
- `DELETE /api/v1/telegram_subscriptions/:id` - Удаление подписки

**Уведомления:**
- `GET /api/v1/telegram_notifications` - История уведомлений
- `POST /api/v1/telegram_notifications` - Отправка уведомления
- `POST /api/v1/telegram_notifications/bulk` - Массовая отправка

**Администрирование:**
- `GET /api/v1/telegram_subscriptions/statistics` - Статистика
- `GET /api/v1/telegram_webhook/config` - Конфигурация бота
- `POST /api/v1/telegram_webhook/test_bot` - Тестирование

## 📝 ИНСТРУКЦИИ ПО ЗАПУСКУ

### Настройка Backend

1. **Установка зависимостей:**
```bash
cd tire-service-master-api
bundle install
```

2. **Настройка переменных окружения:**
```bash
# .env
TELEGRAM_BOT_TOKEN=your_bot_token_here
TELEGRAM_WEBHOOK_URL=https://your-domain.com/api/v1/telegram_webhook
```

3. **Миграции базы данных:**
```bash
rails db:migrate
```

4. **Настройка Telegram бота:**
```bash
rails telegram:setup_bot
rails telegram:set_webhook
```

### Настройка Frontend

1. **Установка зависимостей:**
```bash
cd tire-service-master-web
yarn install
```

2. **Запуск в режиме разработки:**
```bash
yarn start
```

3. **Доступ к функциям:**
- Страница профиля: `/profile/settings`
- Админ-панель: `/admin/telegram-notifications`
- Тестирование: `external-files/testing/html/test_telegram_integration.html`

## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Архитектура компонентов

```typescript
// Основная структура типов
interface TelegramSubscription {
  id: number;
  user_id: number;
  chat_id: string;
  username?: string;
  notification_preferences: TelegramNotificationPreferences;
  status: TelegramSubscriptionStatus;
  // ... другие поля
}

// API интеграция
export const telegramApi = baseApi.injectEndpoints({
  endpoints: (builder) => ({
    getUserTelegramSubscriptions: builder.query<TelegramSubscription[], void>({
      query: () => ({ url: 'telegram_subscriptions', method: 'GET' }),
      providesTags: ['TelegramSubscription'],
    }),
    // ... другие эндпоинты
  }),
});
```

**TelegramIntegration** - Универсальный компонент с вариантами отображения:
- `variant="card"` - Карточка для страницы профиля

**TelegramNotificationManager** - Админ-панель с вкладками:
- Статистика с карточками метрик
- Управление подписками с таблицей
- История уведомлений с фильтрацией
